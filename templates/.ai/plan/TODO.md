# Project TODO — Four Quadrants

> Eisenhower Matrix. Agents update this file during every plan phase.
> Completed items (`[x]`) are archived to `docs/archive/todo-archive-YYYY-MM.md` at the start of each session.

---

## Urgent + Important

_Critical items that block progress or have imminent deadlines._

- [ ] (example) Fix production bug in auth flow (source: @user, created: YYYY-MM-DD)

## Important + Not Urgent

_High-value items for future iterations. Plan and schedule._

- [ ] (example) Refactor config module for extensibility (source: @claude, created: YYYY-MM-DD)

## Urgent + Not Important

_Time-sensitive but low-impact. Delegate or batch._

- [ ] (example) Update CI badge in README (source: @codex, created: YYYY-MM-DD)

## Not Urgent + Not Important

_Nice-to-have. Review periodically, drop if stale._

- [ ] (example) Explore alternative logging library (source: @claude, created: YYYY-MM-DD)
