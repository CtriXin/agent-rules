# Agent Rules

Universal AI collaboration rules extracted from production projects. Technology-agnostic, reusable across any codebase.

## Usage

Copy `AGENT_RULES.md` to your project root as `CLAUDE.md` or `AGENT.md`, or reference it from your existing AI config.

## Files

| File | Purpose |
|------|---------|
| `AGENT_RULES.md` | Complete rule set for AI agents |
| `rules/iteration.md` | Iteration discipline (6-step cycle) |
| `rules/plan-quality.md` | Plan quality gates (preflight + challenge) |
| `rules/code-redlines.md` | Code quality hard limits |
| `rules/error-handling.md` | Error handling protocol |
| `rules/documentation.md` | Documentation sync rules |
| `rules/sensitive-ops.md` | Sensitive operation guardrails |
| `rules/worktree.md` | Git worktree parallel dev rules |
